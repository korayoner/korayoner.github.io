# korayoner.github.io

Koray Öner'in hesaplama araçları sitesi. Tüm araçlar ücretsizdir, üyelik gerektirmez ve reklamsızdır; hesaplamalar tarayıcıda yapılır.

- **[Brüt Net Maaş Hesaplama 2026](https://korayoner.github.io/maas-hesaplama/)** — güncel vergi dilimleri, SGK tavanı ve asgari ücret istisnasıyla 12 aylık bordro dökümü; netten brüte hesaplama, PDF rapor.
- **[Kıdem ve İhbar Tazminatı Hesaplama 2026](https://korayoner.github.io/kidem-tazminati-hesaplama/)** — güncel tavanla gün gün kıdem tazminatı, ihbar tazminatı ve PDF rapor.

Tüm projeler için ana site: [onerkoray.github.io](https://onerkoray.github.io/)
